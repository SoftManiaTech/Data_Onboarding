# Data Onboarding 

Login as root user 
```bash
sudo su
```
Install git
```bash
yum install git -y
```

switch to splunk user
```bash
sudo su - splunk
```
Navigate to tmp folder
```bash
cd /tmp/
```
Clone this repository
```bash
git clone <repo_link>
```


This repo contains the sample data & configurations related to the Data onboarding usecases for Splunk Enterprise
